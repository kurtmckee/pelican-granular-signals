Changelog
*********

Unreleased changes
==================


1.2.0 - 2024-09-09
==================

*   Support Python 3.13.
*   Drop support for Python 3.8.

1.1.0 - 2023-05-22
==================

*   Support Python 3.12 and 3.11.
*   Drop support for Python 3.7 and 3.6.


1.0.1 - 2022-03-31
==================

*   Support Python 3.10.


1.0.0 - 2021-08-08
==================

*   Send five additional signals after Pelican's ``finalized`` signal:
    ``sitemap``, ``optimize``, ``minify``, ``compress``, and ``deploy``.
